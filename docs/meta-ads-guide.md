# 메타 광고 집행 가이드 (화면별)

준비물: 개인 페이스북 계정, 결제 가능한 카드

## 1단계 — 비즈니스 계정 만들기
1. business.facebook.com 접속 → 로그인
2. "비즈니스 계정 만들기" 클릭 → 비즈니스 이름(예: "포항서숲캠핑장" 또는 사업자명), 본인 이름, 이메일 입력 → 제출
3. 좌측 메뉴에서 "광고 계정" → "만들기" → 광고 계정 이름 입력, 시간대(GMT+9 서울), 통화(KRW) 확인 후 생성

## 2단계 — 결제수단 등록
1. 좌측 메뉴 "결제수단" → 카드 추가 → 카드 정보 입력
2. 이 단계에서 실제 카드 등록/결제 정보가 들어가므로 본인이 직접 진행

## 3단계 — 캠페인 만들기 (Ads Manager)
1. Ads Manager (adsmanager.facebook.com) 접속 → "만들기" 버튼
2. 캠페인 목표 선택: **"잠재고객"(리드)** 또는 **"트래픽"** 중 선택
   - 리드: 페이스북 안에서 바로 폼 작성 (이탈 적음, 리드 품질은 낮을 수 있음)
   - 트래픽: 우리 랜딩페이지로 직접 유입 (리드 품질 높음, 이탈 있음)
   - **추천: 트래픽** — 랜딩페이지에서 "실제 운영 사이트" 증거를 직접 보여줘야 설득력이 생기므로
3. 캠페인 이름: "캠핑장_마케팅자동화_1차테스트" 정도로 구분되게 입력

## 4단계 — 예산 설정
1. 예산 유형: "캠페인 총예산" 선택 → 100,000원 입력 (또는 일 예산 10,000원 × 기간 10일)
2. 일정: 시작일 오늘, 종료일 오늘로부터 10~14일 후

## 5단계 — 타겟(오디언스) 설정
1. 위치: 대한민국 전체
2. 연령: 30~65세 (캠핑장 사장님 연령대 고려)
3. 상세 타겟팅(관심사/행동)에 아래 키워드로 검색해서 추가:
   - "캠핑", "글램핑", "오토캠핑", "캠핑장" (관심사)
   - "소상공인" 또는 "중소기업 사업주" 같은 비즈니스 오너 타겟팅 옵션이 있으면 함께 추가 (플랫폼 UI에 따라 "Business decision makers" 유사 카테고리로 표시될 수 있음)
4. 노출 위치: "자동 노출 위치" 유지 (페이스북+인스타그램 피드/릴스 전체에 최적화 노출됨)

## 6단계 — 광고 소재 등록
1. 광고 형식: "단일 이미지" 선택
2. 이미지 업로드: `assets/hero-v3.png` (리포에 있는 파일을 다운로드해서 업로드)
3. 기본 텍스트(본문)/헤드라인: `docs/ad-copy.md`의 버전 A/B/C 중 하나 붙여넣기 (처음엔 버전 A 추천, 예산이 남으면 버전을 나눠서 A/B 테스트)
4. 웹사이트 URL: `https://a01059113805.github.io/camping-marketing-landing/`
5. 행동 유도(CTA) 버튼: "더 알아보기" 선택

## 7단계 — 검토 제출
1. 우측 하단 "게시" 클릭
2. 메타 검토(보통 몇 시간~24시간 이내) 통과하면 자동으로 게재 시작

## 8단계 — 성과 확인 (게재 시작 후)
1. Ads Manager 캠페인 목록에서 클릭 수, 도달, 지출 확인
2. 실제 문의는 구글폼 응답으로 확인: https://docs.google.com/forms/d/13RrDGW_hAk10CAt5vSSaMCfHDH0df6-lFrgwu4bwtZk/edit (편집 화면 → "응답" 탭)
