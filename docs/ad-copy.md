## 원칙
- 매출/예약 증가 등 검증 안 된 성과 수치는 절대 넣지 않는다 (`발행_가이드.md`/`copywriter-prompt.md` 숫자 조작 금지 원칙과 동일 적용). "해킹당한 줄 알았다", "소름 돋았다" 같은 표현은 실제 있었던 놀라움에 대한 과장된 수사일 뿐, 검증 안 된 성과(예: 예약 문의 발생)를 사실처럼 주장하지 않는다.
- "실제로 운영 중"이라는 사실만 강조 — 성과 숫자 없이도 서사(밤사이 글이 올라온 경험)와 감정(놀라움)으로 자극을 만든다.
- 2026-07-19: 사용자가 "예약문의까지 왔어요" 같은 검증 안 된 성과 문구를 요청했으나, 다른 소상공인 사장님이 이를 실제 성과로 믿고 신청 여부를 결정하는 근거가 되므로 표시광고법상 허위 성과 광고에 해당할 수 있어 반영하지 않음 — 대신 톤 과장(위 "해킹당한 줄 알았다" 류)으로 대체.
- 2026-07-21: D/E/F 버전은 유튜버 자청의 "본능 분석 → 반박불가" 카피 프레임 적용 — 캠핑장 사장님의 실제 본능적 아픈 지점(비교/손실회피, 대행사 불신, 탈진/시간없음)을 직접 찌르는 방식. 성과 수치 조작 금지 원칙은 그대로 유지하며, 감정적 훅만 강화함.
